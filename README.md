# Snake Game - Python Tkinter Project 🐍

## Project Overview

This project is a GUI-based Snake Game developed using Python and
Tkinter. It is designed as a mini-project to demonstrate concepts of GUI
programming, event handling, and game logic implementation.

------------------------------------------------------------------------

## Features

-   Interactive Snake Game Interface
-   Score Tracking System
-   Start Game Button
-   Restart Game Button
-   Keyboard Controls (Arrow Keys)
-   Clean and User-Friendly GUI

------------------------------------------------------------------------

## Technologies Used

-   Python
-   Tkinter (Python GUI Library)
-   Random Module

------------------------------------------------------------------------

## Project Structure

Snake Game/ │ ├── snake_beautiful.py ├── snake_console.py ├──
snake_gui.py └── README.md

------------------------------------------------------------------------

## How to Run the Project

Step 1: Install Python (if not already installed)

Step 2: Open terminal or command prompt

Step 3: Navigate to project folder

Step 4: Run the game using:

python snake_beautiful.py

------------------------------------------------------------------------

## Controls

Arrow Keys → Move Snake Start Button → Begin Game Restart Button →
Restart Game
